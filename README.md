This is a simple and clean Todo List application built using React, TypeScript, TailwindCSS, and Vite.
You can add tasks, delete them, and all data is saved in localStorage.
## Features
- Add new tasks
- Delete tasks
- LocalStorage persistence
- Fully styled with TailwindCSS
- Strong typing with TypeScript
- Clean, scalable component structure
## Tech Stack
- React 18
- TypeScript
- TailwindCSS 3.4
- Vite
- LocalStorage API
## Installation
Clone the repository:
git clone https://github.com/Asall-/todo-ts-react-1st-try.git
cd todo-ts-react
Install dependencies:
npm install
Run the projec:
npm run dev
## Tailwind Setup
TailwindCSS is fully configured in this project.
## How It Works
- Tasks are stored in React state.
- Whenever tasks change, updated data is saved to localStorage.
- On initial load, data is read from localStorage and restored.
